# Homebrew Tap for Warden

## Install

```bash
brew install --cask ParthJadhav/warden/warden
```

Or:

```bash
brew tap ParthJadhav/warden
brew install --cask warden
```

## About

[Warden](https://github.com/ParthJadhav/Warden-app) is a macOS menu bar utility for screen locking. Requires macOS 15 (Sequoia) or later.
